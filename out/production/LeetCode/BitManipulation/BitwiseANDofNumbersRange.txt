给定一个范围，要求输出这个范围内所有数的 AND 最后的结果（inclusive）
0 <= m <= n <= 2147483647

果然暴力解会 TLE：m = 0, n = 2147483647

那就是要找规律了
瞬间想到 n & (n - 1)，但是不清楚这里怎么用哈哈
如果 m 为 0，那么肯定 AND 肯定为 0

直接看答案吧

看了答案瞬间觉得好简单哈哈

将 32 位上的问题拆分成每一位上的子问题
在任何一位上如果出现 0，那么这一位上最后的结果肯定是 0
所以如果 m 不等于 n，那么必然存在最低位不等，也就是说 m -> n 这个范围中必然会出现最低位为 0 的情况

用 n & (n - 1) 更简单
只要 n 还大于 m，就让 n 的最低位 1 置为 0
