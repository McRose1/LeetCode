Matrix 题，但其实考察的是二分法

每一行递增
并且每一个元素也是递增的
binary search 的变形题

看看能不能把 matrix 拉直成一维数组
可以是可以，只是两个 for 循环要 n^2 相当于总时间复杂度为 n^2 + logn

应该是得直接在矩阵里进行 binary search 
那第一步肯定是先得确定 target 在哪一行，只要确定了行数，后面就是普通的 binary search

自己写的版本在确定哪一行的时候用到是暴力解，但其实确定行数也可以使用二分法，主要思想也是通过每一行的最后一个元素来划分界限

其实这题还有更简单，就是上面说的一维数组，但是不是“拉成”，而是把矩阵看作是有序的一维数组
数组的 index 可以转换为 matrix 上对应的坐标：
row = idx / n; col = idx % n

这样就变成了一道普通的二分法搜索，比先对行进行二分，在对列进行二分要方便
