## Sort 
我们发现，奇数位的数字大于两侧偶数位的数字，也就是说，最后的数组，只要是奇数位，那么必定大于前面的偶数位；如果是偶数位，那么必定小于前面的奇数位

if ((i % 2 == 1 && nums[i] < nums[i - 1]) || (i % 2 == 0 && nums[i] > nums[i - 1])) -> swap
