山景城一姐
对于有向图，
每个顶点有两个“度”
从该点出发的边数为“出度”
指向该点的边数为“入度”

这个题我们只关心入度

   0(0)  
     \ 
      4(2)  
     /   \
  1(0)     5(2) 
    \    /
     3(2)  
    /
  2(0)
  
先根据课表做一个 graph
在程序里面表示 graph 的常见两种方式：
1. 邻接矩阵（adjancency list）
2. 邻接表（adjancency graph）HashMap

这个题用邻接表
key 是当前 node
value 是从这个 node 能到达的 node

同时维护一个 inDegree 数组作为限制

有了图以后就能开始获取拓扑序列，相当于用 BFS/DFS
