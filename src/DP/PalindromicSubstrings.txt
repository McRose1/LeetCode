# 647. Palindromic Substrings
从字符串中计算所有回文子字符串的数量

## DP
要注意这题的 DP 数组不是为了得到最后的 count，并不是 int[][]，而是 boolean[][]，相当于代替了 isPalindrome() 的功能，将判断回文字符串时间复杂度从 O(n) -> O(1)，只是辅助而不是存最终答案

    0 1 2 3 4
    x a b a x
0 x T F F F T
1 a   T F T F
2 b     T F F 
3 a       T F
4 x         T

DP 数组一画出来就知道了，相当于就是有多少个 True，答案就是多少

注意对于长度为 2 的子字符串的判断，不再适用于 dp[i][j] = dp[i + 1][j - 1]，因为 i 相当于字符串的左边界，j 相当于字符串的右边界，如果长度小于 3，那么[i, j]表示的字符串是不合法的

## 中心向外扩散法
这个方法比较 tricky，主要是找回文字符串的中心，比如奇数个的回文字符串是中间那个元素，偶数个是中间 2 个元素中间的那个点
