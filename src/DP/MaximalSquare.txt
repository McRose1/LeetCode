Brute Force: enumerate all top-left corner and try all possible sizes 

dp[x][y] = max size can achieve at (x, y) as bottom right 

dp[x][y] = 
                0                                                    if matrix[x][y] == 0
min(dp[x - 1][y], dp[x][y - 1], dp[x- 1][y - 1]) + 1                 if matrix[x][y] == 1 
主要就是看左上角、右上角和对角线的正方形
